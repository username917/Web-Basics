This project requires the following:

Make a new React project, using "npx create-react-app my-bootstrap-project"

Once created, install "bootstap" and "react-router-dom" from a new terminal within VS Code or other IDE

Copy the components folder under "src", so it becomes "components" and replace the App.js file in the project
with the one included here.

Project Details:

This project is a demonstration of how to implement conditional rendering with React. The "Home.js" file has 
the comprehensive code on how to achieve that with a dropdwon of fruit choices where choosing "banana" 
output a heading and a new button.

This is going to be a project in progress.
